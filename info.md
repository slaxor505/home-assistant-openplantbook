# 🌿 OpenPlantbook

Connects Home Assistant to the [OpenPlantbook API](https://open.plantbook.io/) — search plant species, fetch care data, and upload sensor readings.

Used as the data backend for the [Plant Monitor](https://github.com/slaxor505/homeassistant-plant) integration.

## 📖 Documentation

See the [README](https://github.com/slaxor505/home-assistant-openplantbook/) for setup instructions, configuration options, and action examples.
